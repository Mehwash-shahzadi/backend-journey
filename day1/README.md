# Day 1: Backend Journey Setup

A simple Python script to check the environment, compute factorial, and test `rich` for enhanced console output and `requests` for making HTTP requests.

## Project Structure

backend_journey/
└── day1/
├── setup_check.py
├── output.txt
├── README.md

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Mehwash-shahzadi/backend_journey.git
   cd backend_journey/day1
   ```
2. Create and activate a virtual environment:
   ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```
3. Install dependencies:
   ```bash
    pip install -r requirements.txt
   ```
4. Run the setup check script:
   ```bash
    python setup_check.py
   ```
5. Check the output in `output.txt` for results.
